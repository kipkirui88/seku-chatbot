How the Bot works.

Option 1.

This is a Whatsapp chatbot which allows SEKU students pay their school fees. To start the process:

  • Click on Whatsapp Icon on the right, bottom side.
  • You'll be refered to whatsapp with already typed message "Join necessary-visitor".
  • Click on send button and you'll be ready to chat with the bot.
  • Start chatting with the bot by sending message "Hi".
  • Follow the option provided, and remember to start your phone number with country code "254" for Kenya.
  • Enter phone number you would pay fees with it & you'll receive a Mpesa STK push prompt to enter your Pin.
